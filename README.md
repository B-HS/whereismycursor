## Install

```bash
git clone https://github.com/hyunseokbyun/whereismycursor
cd whereismycursor
./build-app.sh release
open .build/CursorSpotlight.app
```

Requires macOS 13 or later.

## Usage

1. The app lives in the menu bar (no Dock icon).
2. Press and hold the hotkey — default `Command + Option + F1`.
3. Release to dismiss.

Click the menu bar icon and choose **Settings** to change the shortcut, overlay color, radius, language, and more.
