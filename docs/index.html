Hold. Spot. Release.

A zero-latency macOS utility that spotlights your mouse cursor. Bind a hotkey — while you hold it, the rest of the screen dims. Built in pure Swift + AppKit. No Electron. No bloat.

[ 02 ] Features

What it does.

/ 01

Hold-to-Show

Press your bound hotkey; the overlay appears instantly. Release it; gone. No toggle state to remember.

/ 02

Custom Shortcut

Record any key combination directly in the settings window. Single keys, full modifier stacks — both work.

/ 03

Color & Pattern

Pick overlay color with alpha, adjust master opacity, or layer an image pattern as the mask fill.

/ 04

Multi-Monitor

One NSPanel per screen at .screenSaver level. Works across Spaces, full-screen apps, and display hot-swap.

/ 05

Trilingual UI

English · 한국어 · 日本語. Switch in-app with no restart. Follows system locale by default.

/ 06

Launch at Login

SMAppService-based. Toggle once, it boots with your Mac. Settings persist via UserDefaults.

[ 03 ] Specs

Under the hood.

Native AppKit + Core Animation. The overlay is a single CAShapeLayer with evenOdd fill — one rectangle minus one ellipse. Per-screen NSPanel at .screenSaver level with ignoresMouseEvents. Global hotkeys via Carbon, not NSEvent — no Accessibility permission prompt needed. 

// package & runtime
language      Swift 6
target        macOS 13.0+
ui            SwiftUI (settings) + AppKit (overlay)
size          416 KB signed .app
deps          HotKey (single dep)

// overlay pipeline
window        NSPanel · .screenSaver · nonactivating
layer         CAShapeLayer · evenOdd
tracking      NSEvent global + local monitors
mouse         ignoresMouseEvents = true

// persistence
settings      UserDefaults
login item    SMAppService.mainApp
i18n          .lproj bundles, runtime swap
